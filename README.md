# pinkimac
2026年5月に研究室に導入された ピンクiMac G4 運用の情報共有リポジトリ


## 作業メモ
* [Issues](https://github.com/furuhashilab/pinkimac/issues) に記録


## 運用ルール
* 各個人が自由に使えるディスク容量は Max 20GB/アカウント( 20GB x 20 =  ) までとする。それ以上のディスク容量を必要とする場合は個別に外部SSDなどを接続して作業を行うこと。
   - iCloude同期に気をつけること。特に写真や動画、ドキュメントなどは同期オフを推奨とする。
   - pinkiMac マシンの空き容量は 500GB を維持するように全員でチェックする。仮に残り 300GB となった場合は運用ルールを修正する。

* 卒業後６か月(毎年9月30日迄)はアカウントは保持するが、それ以降はアカウントは削除する。

* Python などメジャーなツールの動作環境で環境汚染が予想される場合は適宜、研究室のInstagramグループチャット内で情報共有を行う。

* 可能な限り [Issues](https://github.com/furuhashilab/pinkimac/issues) に記録する。

* 深刻なトラブルが発生した場合３日以内に Issue として必ず報告すること。これを怠った場合は pinkiMac アカウント保持者全員に飯をおごること :-)
